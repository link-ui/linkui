    快速上手

    请查看文档并且按照步骤开始使用

    脚手架

    如果你正在使用Vue cli vue-cli@3，推荐使用 Vue 官方提供的 脚手架 
    //安装vue-cli
npm install -g @vue/cli

//创建一个自己的项目
vue create mydemo

    创建完成后,打开图片化界面

    //打开图片化界面
npm run dev

    安装 npm

    npm install linkui

    引入组件

    import Vue from "vue";
import LinkUI from "linkui";
import "linkui/lib/index.css";
Vue.use(LinkUI)
